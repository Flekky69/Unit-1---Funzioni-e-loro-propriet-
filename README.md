# Flashcard: UnitĂ  1 - Funzioni e loro ProprietĂ 

| Fronte (Domanda / Concetto) | Retro (Risposta / Definizione / Formula) |
| :--- | :--- |
| **Definizione di Funzione reale di variabile reale** | Dati due sottoinsiemi non vuoti $A$ e $B$ di $\mathbb{R}$, una funzione $f$ da $A$ a $B$ Ăš una relazione che associa a *ogni* numero reale di $A$ *uno e un solo* numero reale di $B$. |
| **Classificazione delle funzioni algebriche** | Una funzione si dice **algebrica** se l'espressione analitica $y = f(x)$ contiene solo, per la variabile $x$, operazioni di addizione, sottrazione, moltiplicazione, divisione, elevamento a potenza o estrazione di radice.<br>Si dividono in:<br>1. **Razionale intera (o polinomiale):** espressa da un polinomio (lineare se di 1Â° grado, quadratica se di 2Â° grado).<br>2. **Razionale fratta:** espressa mediante quozienti di polinomi.<br>3. **Irrazionale:** se la variabile $x$ compare sotto il segno di radice. |
| **Definizione di Funzione Trascendente** | Una funzione $y = f(x)$ che non Ăš algebrica (es. logaritmica, esponenziale, goniometrica). |
| **Definizione di Dominio Naturale (o Campo di Esistenza)** | Ă l'insieme piĂč ampio dei valori reali che si possono assegnare alla variabile indipendente $x$ affinchĂ© esista il corrispondente valore reale $y$. |
| **Dominio delle Funzioni Irrazionali**<br>$$y = \sqrt[n]{f(x)}$$ | * Se $n$ Ăš **pari**: $\\{x \in \mathbb{R} \mid f(x) \ge 0\\}$<br>* Se $n$ Ăš **dispari**: coincide con il dominio di $f(x)$. |
| **Dominio delle Funzioni Logaritmiche**<br>$$y = \log_a f(x) \quad (a > 0, a \neq 1)$$|$$\\{x \in \mathbb{R} \mid f(x) > 0\\}$$ |
| **Dominio delle Funzioni Esponenziali**<br>1. $y = a^{f(x)}$<br>2. $y = [f(x)]^{g(x)}$<br>3. $y = f(x)^\alpha$ ($\alpha$ irrazionale) | 1. Coincide con il dominio di $f(x)$.<br>2. $\\{x \in \mathbb{R} \mid f(x) > 0\\} \cap \text{dominio di } g(x)$.<br>3. $\\{x \in \mathbb{R} \mid f(x) \ge 0\\}$ se $\alpha > 0$; $\\{x \in \mathbb{R} \mid f(x) > 0\\}$ se $\alpha < 0$. |
| **Dominio di Tangente e Cotangente**<br>1. $y = \tan x$<br>2. $y = \cot x$ | 1. $\mathbb{R} - \left\\{\frac{\pi}{2} + k\pi\right\\}, \text{ con } k \in \mathbb{Z}$<br>2. $\mathbb{R} - \\{k\pi\\}, \text{ con } k \in \mathbb{Z}$ |
| **Dominio di Arcoseno e Arcocoseno**<br>$y = \arcsin x$ e $y = \arccos x$ | $$[-1; 1]$$ |
| **Definizione di Zero di una funzione** | Un numero reale $a$ Ăš uno zero della funzione $y = f(x)$ se:$$f(a) = 0$$Graficamente rappresenta un punto di intersezione della funzione con l'asse $x$. |
| **Funzione Iniettiva** | Una funzione da $A$ a $B$ Ăš iniettiva se ogni elemento di $B$ Ăš immagine di **al piĂč un** elemento di $A$. |
| **Funzione Suriettiva** | Una funzione da $A$ a $B$ Ăš suriettiva se ogni elemento di $B$ Ăš immagine di **almeno un** elemento di $A$ (cioĂš il codominio coincide con $B$). |
| **Funzione Biunivoca (o Biiettiva)** | Una funzione che Ăš **sia iniettiva sia suriettiva**. |
| **Funzione Crescente in senso stretto** | $y = f(x)$ di dominio $D \subseteq \mathbb{R}$ Ăš crescente in senso stretto in un intervallo $I \subseteq D$ se, comunque scelti $x_1$ e $x_2$ in $I$:$$\text{con } x_1 < x_2 \implies f(x_1) < f(x_2)$$ |
| **Funzione Decrescente in senso stretto** | $y = f(x)$ di dominio $D \subseteq \mathbb{R}$ Ăš decrescente in senso stretto in un intervallo $I \subseteq D$ se, comunque scelti $x_1$ e $x_2$ in $I$:$$\text{con } x_1 < x_2 \implies f(x_1) > f(x_2)$$ |
| **Definizione di Funzione MonĂČtona** | Una funzione Ăš monĂČtona in senso stretto in un intervallo $I$ se in quell'intervallo Ăš **sempre crescente** o **sempre decrescente** in senso stretto. |
| **Funzione Pari** | Una funzione $y = f(x)$ di dominio $D$ (simmetrico rispetto all'origine) Ăš pari se per qualunque $x \in D$:$$f(-x) = f(x)$$Il suo grafico Ăš simmetrico rispetto all'**asse $y$**. |
| **Funzione Dispari** | Una funzione $y = f(x)$ di dominio $D$ (simmetrico rispetto all'origine) Ăš dispari se per qualunque $x \in D$:$$f(-x) = -f(x)$$Il suo grafico Ăš simmetrico rispetto all'**origine degli assi**. |
| **ProprietĂ  geometriche delle Funzioni Esponenziali**<br>$$y = a^x$$ | * **Dominio:** $\mathbb{R}$; **Immagine:** $\mathbb{R}^+$ (se $a \neq 1$). Il grafico sta tutto sopra l'asse $x$.<br>* **Intersezioni:** Non interseca l'asse $x$, interseca l'asse $y$ in $(0; 1)$.<br>* **Monotonia:** Se $a > 1$ Ăš sempre crescente; se $0 < a < 1$ Ăš sempre decrescente; se $a = 1$ Ăš costante ($y = 1$). |
| **ProprietĂ  geometriche delle Funzioni Logaritmiche**<br>$$y = \log_a x$$ | * **Dominio:** $\mathbb{R}^+$; **Immagine:** $\mathbb{R}$.<br>* **Intersezioni:** Interseca l'asse $x$ in $(1; 0)$, non interseca l'asse $y$.<br>* **Monotonia:** Se $a > 1$ Ăš sempre crescente; se $0 < a < 1$ Ăš sempre decrescente. |
| **ProprietĂ  di Seno e Coseno**<br>1. $y = \sin x$<br>2. $y = \cos x$ | * **Dominio:** $\mathbb{R}$; **Immagine:** $[-1; 1]$.<br>* **PeriodicitĂ :** Periodiche di periodo $2\pi$.<br>* **Simmetria:** Il seno Ăš **dispari** ($\sin(-x) = -\sin x$), il coseno Ăš **pari** ($\cos(-x) = \cos x$).<br>* **Crescienza:** $\sin x$ cresce in $\left[-\frac{\pi}{2} + 2k\pi; \frac{\pi}{2} + 2k\pi\right]$; $\cos x$ cresce in $[-\pi + 2k\pi; 0 + 2k\pi]$. |
| **ProprietĂ  di Tangente e Cotangente**<br>1. $y = \tan x$<br>2. $y = \cot x$ | * **Immagine:** $\mathbb{R}$.<br>* **Simmetria:** Entrambe sono funzioni **dispari** ($\tan(-x) = -\tan x$, $\cot(-x) = -\cot x$).<br>* **PeriodicitĂ :** Periodiche di periodo $\pi$.<br>* **Monotonia:** $\tan x$ Ăš strettamente crescente in $]-\frac{\pi}{2} + k\pi; \frac{\pi}{2} + k\pi[$; $\cot x$ Ăš strettamente decrescente in $]0 + k\pi; \pi + k\pi[$. |
| **Definizione di Funzione Inversa** | Data una funzione **biunivoca** $y = f(x)$ da $A$ a $B$, la funzione inversa $f^{-1}$ Ăš la funzione biunivoca $x = f^{-1}(y)$ da $B$ a $A$ che associa a ogni $y$ di $B$ il valore $x$ di $A$ tale che $y = f(x)$. |
| **Condizione di InvertibilitĂ ** | Una funzione ammette inversa se e solo se Ăš **biunivoca** (biiettiva). Le funzioni strettamente monĂČtone su un intervallo sono iniettive, quindi invertibili se si considera come codominio il loro insieme immagine. Se non Ăš biunivoca, si puĂČ operare una **restrizione del dominio**. |
| **Simmetria del grafico di una funzione inversa** | Il grafico di una funzione e quello della sua inversa sono **simmetrici rispetto alla bisettrice del primo e del terzo quadrante** ($y = x$). |
| **Definizione di Funzione Composta** | Date due funzioni $f$ e $g$, la funzione composta $g \circ f$ (si legge "$g$ composto $f$") Ăš definita come:$$y = g[f(x)]$$Si ottiene associando a ogni elemento $x$ del dominio di $f$ (tale che l'immagine $f(x)$ appartenga al dominio di $g$) il valore $y$ immagine di $f(x)$ mediante $g$. |
| **ProprietĂ  Commutativa nella composizione di funzioni** | In generale, la composizione di funzioni **non Ăš commutativa**:$$g \circ f \neq f \circ g$$ |
